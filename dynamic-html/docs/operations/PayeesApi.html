This is the API

getPayeeDetail

Obtain detailed information on a single payee

URL

https://data.provider.com.au/cds-au/v1/banking/payees/{payeeId}

HTTP Method

GET

Response Type

ResponseBankingPayeeDetails

Parameters

  • Path: payeeId String

    The ID used to locate the details of a particular payee

listPayees

Obtain a list of pre-registered payees

URL

https://data.provider.com.au/cds-au/v1/banking/payees

HTTP Method

GET

Response Type

ResponseBankingPayees

Parameters

  • Query: type String

    Filter on the payee type field. In addition to normal type field values, ALL can be specified to retrieve all payees. If absent the assumed value is ALL

  • Query: page Integer

    Page of results to request (standard pagination)

  • Query: page-size Integer

    Page size to request. Default is 25 (standard pagination)
