This is the API

getAccountDetail

Obtain detailed information on a single account

URL

https://data.provider.com.au/cds-au/v1/banking/accounts/{accountId}

HTTP Method

GET

Response Type

ResponseBankingAccount

Parameters

  • Path: accountId String

    A tokenised identifier for the account which is unique but not shareable

getTransactionDetail

Obtain detailed information on a transaction for a specific account

URL

https://data.provider.com.au/cds-au/v1/banking/accounts/{accountId}/transactions/{transactionId}

HTTP Method

GET

Response Type

ResponseBankingTransactionDetail

Parameters

  • Path: accountId String

    The account id token that is used to uniquely represent the account

  • Path: transactionId String

    The unique identifier for the specific transaction for which details are being requested

getTransactions

Obtain transactions for a specific account

URL

https://data.provider.com.au/cds-au/v1/banking/accounts/{accountId}/transactions

HTTP Method

GET

Response Type

ResponseBankingTransactions

Parameters

  • Path: accountId String

    ID of the account to get transactions for. Must have previously been returned by one of the account list end points.

  • Query: start-time String

    Constrain the transaction history request to transactions with effective time at or after this date/time. If absent defaults to current time. Format is aligned to DateTimeString common type

  • Query: end-time String

    Constrain the transaction history request to transactions with effective time at or before this date/time. If absent defaults to start-time plus 100 days. Format is aligned to DateTimeString common type

  • Query: min-amount String

    Filter transactions to only transactions with amounts higher or equal to than this amount

  • Query: max-amount String

    Filter transactions to only transactions with amounts less than or equal to than this amount

  • Query: text String

    Filter transactions to only transactions where this string value is found as a substring of either the reference or description fields. Format is arbitrary ASCII string

  • Query: page Integer

    Page of results to request (standard pagination)

  • Query: page-size Integer

    Page size to request. Default is 25 (standard pagination)

listAccounts

Obtain a list of accounts

URL

https://data.provider.com.au/cds-au/v1/banking/accounts

HTTP Method

GET

Response Type

ResponseBankingAccounts

Parameters

  • Query: open-status String

    Used to filter results according to open/closed status. Values can be OPEN, CLOSED or ALL. If absent then ALL is assumed

  • Query: is-owned String

    Filters accounts based on whether they are owned by the authorised customer

  • Query: product-category String

    Used to filter results on the productCategory field applicable to accounts. Any one of the valid values for this field can be supplied. If absent then all accounts returned.

  • Query: page Integer

    Page of results to request (standard pagination)

  • Query: page-size Integer

    Page size to request. Default is 25 (standard pagination)
