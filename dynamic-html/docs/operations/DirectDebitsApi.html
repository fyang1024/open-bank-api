This is the API

listDirectDebits

Obtain direct debit authorisations for a specific account

URL

https://data.provider.com.au/cds-au/v1/banking/accounts/{accountId}/direct-debits

HTTP Method

GET

Response Type

ResponseBankingDirectDebits

Parameters

  • Path: accountId String

    ID of the account to get direct debit authorisations for. Must have previously been returned by one of the account list end points.

  • Query: page Integer

    Page of results to request (standard pagination)

  • Query: page-size Integer

    Page size to request. Default is 25 (standard pagination)

listDirectDebitsBulk

Obtain direct debit authorisations for multiple, filtered accounts

URL

https://data.provider.com.au/cds-au/v1/banking/accounts/direct-debits

HTTP Method

GET

Response Type

ResponseBankingDirectDebits

Parameters

  • Query: is-owned String

    Filters accounts based on whether they are owned by the authorised customer

  • Query: product-category String

    Used to filter results on the productCategory field applicable to accounts. Any one of the valid values for this field can be supplied. If absent then all accounts returned.

  • Query: page Integer

    Page of results to request (standard pagination)

  • Query: page-size Integer

    Page size to request. Default is 25 (standard pagination)
