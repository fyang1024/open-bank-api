This is the API

getProductDetail

Obtain detailed information on a single product offered openly to the market

URL

https://data.provider.com.au/cds-au/v1/banking/products/{productId}

HTTP Method

GET

Response Type

ResponseBankingProduct

Parameters

  • Path: productId String

    ID of the specific product requested

listProducts

Obtain a list of products that are currently openly offered to the market

URL

https://data.provider.com.au/cds-au/v1/banking/products

HTTP Method

GET

Response Type

ResponseBankingProducts

Parameters

  • Query: effective String

    Allows for the filtering of products based on whether the current time is within the period of time defined as effective by the effectiveFrom and effectiveTo fields. If absent defaults to 'CURRENT'

  • Query: updated-since String

    Only include products that have been updated after the specified date and time. If absent defaults to include all products

  • Query: brand String

    Filter results based on a specific brand

  • Query: product-category String

    Used to filter results on the productCategory field applicable to accounts. Any one of the valid values for this field can be supplied. If absent then all accounts returned.

  • Query: page Integer

    Page of results to request (standard pagination)

  • Query: page-size Integer

    Page size to request. Default is 25 (standard pagination)
